Essential commands
==================
tmux  //To create a session.
ctrl + b + %  //Split vertically.
ctrl + b + "  //Split horizontally.
ctrl + b + <arrow keys>  //To move between panes.
ctrl + b + d  //Detach current session
tmux ls  //List all sessions.
tmux attach-session -t <session number>  //Attach the corresponding session.
exit  //To close the current pane. Exiting all panes kills the corresponding session.
tmux kill-server  //Kills all sessions.
