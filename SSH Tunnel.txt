SSH Tunnel to re-route internet traffic
=======================================
ssh -D 8080 -N -q <username of remote server>@<public IP of remote server> -p <public SSH port of remote server>

For Chrome/Chromium-based Browsers
----------------------------------
Option 1: Use a system-wide proxy
Windows:
Go to Settings > Network & Internet > Proxy.
Under Manual proxy setup, enable Use a proxy server, and set:
Address: localhost
Port: 8080
macOS:
Go to System Preferences > Network > Advanced > Proxies.
Check SOCKS Proxy, set host localhost, port 8080.
Linux (GNOME):
Go to Settings > Network > Network Proxy
Select Manual, SOCKS Host: localhost, Port: 8080

Option 2: Use a proxy extension like FoxyProxy
Install FoxyProxy from the Chrome Web Store.
Add a new proxy configuration:
Proxy Type: SOCKS5
Host: localhost
Port: 8080

For Firefox
-----------
Go to Settings > Network Settings > Settings... > Settings.
Select Manual proxy configuration.
Set:
SOCKS Host: 127.0.0.1
Port: 8080
SOCKS Version: 5
